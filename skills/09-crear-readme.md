# Skill: Crear README

## Objetivo

Crear o actualizar README.md del proyecto IMPULSO DIGITAL.

## Debe explicar

- Qué es el proyecto.
- Estructura de carpetas.
- Qué archivos son públicos.
- Qué archivos son administrativos.
- Cómo entrar al login.
- Cómo funciona la sesión temporal.
- Cómo configurar Supabase.
- Qué tablas se necesitan.
- Qué falta implementar.
- Cómo probar el proyecto localmente.
- Cómo subirlo a hosting.

## Reglas

- Usar lenguaje simple y directo.
- No hacerlo enorme.
- No inventar funcionalidades que todavía no existen.
- Separar lo implementado de lo pendiente.

## Resultado esperado

Un README claro para seguir desarrollando el proyecto sin perderse.
