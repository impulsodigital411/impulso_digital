# Skill: Crear dashboard administrativo

## Objetivo

Crear el dashboard administrativo de IMPULSO DIGITAL.

## Archivo principal

admin/dashboard.html

## Debe incluir

- Sidebar lateral.
- Header superior.
- Nombre: Panel Administrativo IMPULSO DIGITAL.
- Cards de resumen.
- Menú lateral.

## Cards sugeridas

- Consultas recibidas.
- Consultas pendientes.
- Consultas respondidas.
- Servicios activos.

## Menú

- Dashboard.
- Consultas.
- Servicios.
- Categorías.
- Cerrar sesión.

## Archivos relacionados

Lógica:

js/dashboard.js

Estilos:

css/admin.css

## Reglas

- Proteger la página para que no se pueda entrar sin sesión.
- Si no hay sesión, redirigir a admin/login.html.
- El botón cerrar sesión debe borrar la sesión y volver al login.
- No mezclar lógica del dashboard con el sitio público.

## Resultado esperado

Un dashboard visualmente profesional y preparado para cargar datos reales.
