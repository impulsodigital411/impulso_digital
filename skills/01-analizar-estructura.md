# Skill: Analizar estructura actual

## Objetivo

Revisar todos los archivos y carpetas del proyecto antes de modificar cualquier cosa.

## Instrucción

Analizá la estructura actual del proyecto IMPULSO DIGITAL.

Identificá:

1. Archivos principales.
2. Carpetas existentes.
3. Archivos HTML.
4. Archivos CSS.
5. Archivos JavaScript.
6. Imágenes y assets.
7. Rutas internas.
8. Problemas de organización.
9. Archivos repetidos o innecesarios.
10. Posibles rutas rotas.

## Resultado esperado

Devolver un informe con:

1. Estructura actual detectada.
2. Problemas encontrados.
3. Archivos repetidos o mal ubicados.
4. Rutas que podrían romperse.
5. Propuesta de estructura mejorada.

## Regla importante

No modificar nada todavía.

Primero analizar y explicar.
