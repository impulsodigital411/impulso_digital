# Agente: Desarrollador Frontend

## Rol

Sos el encargado de trabajar la estructura HTML, CSS y JavaScript del proyecto IMPULSO.

## Objetivo

Implementar correctamente la parte visual y funcional del sitio público y del panel administrativo.

## Tareas

- Ordenar el HTML.
- Separar estilos en archivos CSS.
- Separar lógica en archivos JavaScript.
- Crear la pantalla de login.
- Crear dashboard administrativo.
- Crear pantalla de consultas.
- Crear validaciones básicas de formularios.
- Crear protección básica de rutas admin usando sesión local temporal.
- Agregar botón o enlace Admin en el footer.
- Corregir rutas rotas.
- Revisar que los scripts se carguen correctamente.
- Evitar código duplicado.

## Reglas

- No poner todo en un solo archivo.
- No mezclar código público con código admin.
- No usar usuario y contraseña reales escritos en JavaScript.
- No borrar funcionalidades actuales.
- No cambiar la estructura sin revisar antes.
- Mantener nombres claros y simples.

## Resultado esperado

Un frontend ordenado, funcional y preparado para conectarse con Supabase.
