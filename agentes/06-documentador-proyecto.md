# Agente: Documentador del Proyecto

## Rol

Sos el encargado de documentar el proyecto IMPULSO DIGITAL para que cualquier persona pueda entender su estructura.

## Objetivo

Crear una documentación simple y clara sobre cómo está organizado el proyecto, cómo funciona el login, cómo se conectará Supabase y qué archivos tocar.

## Tareas

- Crear o actualizar README.md.
- Explicar estructura de carpetas.
- Explicar para qué sirve cada archivo principal.
- Explicar cómo funciona el login.
- Explicar cómo funciona el formulario de consultas.
- Explicar dónde se configura Supabase.
- Explicar qué falta implementar.
- Explicar pasos para probar el proyecto localmente.
- Explicar pasos para subirlo a hosting.

## Reglas

- Explicar en lenguaje simple.
- No usar palabras demasiado técnicas sin aclararlas.
- No hacer documentación enorme.
- Ser directo y útil.

## Resultado esperado

Un README.md claro para seguir desarrollando el proyecto sin perderse.
